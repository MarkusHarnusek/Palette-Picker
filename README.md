# Palette-Picker

Palette Picker is a free and easy to use tool to save your color palettes for your creative projects.

## What Palette-Picker can do

 - Copy the color from a palette, simpy by clicking on it
 - Create random color palettes
 - Edit each color of the color palette with the built-in color picker
 - Save your color palettes
 - Load your color palettes

## What are color palettes like in Palette-Picker like

Color palettes consist of 5 colors; Primary 1, Primary 2, Seconadry 1, Secondary 2 and Text. Each of those colors can be edited using the built-in color picker

## What is the built-in color picker

Palette-Picker's built-in color picker provides a easy to use solution to picking colors. Manually adjust hue, saturation and luminance, while getting a __live__ preview of the color you're workin on. If you don't like playing it with values too much you can just can input a hex color.
